Remember that when two nucleotide chains are bound together by hydrogen bonds, the two nucleotide chains have an antiparallel arrangement, meaning that the 5 end of one molecule is bound to the 3 end of the other molecule, and vice versa.  Also remember that, when forming an RNA molecule using a DNA molecule as a template,
an A RNA nucleotide will base pair with a T DNA nucleotide;
a G RNA nucleotide will base pair with a C DNA nucleotide;
a C RNA nucleotide will base pair with a G DNA nucleotide;
a U RNA nucleotide will base pair with an A DNA nucleotide

So,

Original molecule = 3 C C T A G G C T G C A A T C C 5'
New, RNA primer = --------------- 5 A C G U U A G G 3

When we say that the genetic code is almost universal, what we mean is that almost all organisms use the same codons to specify the same amino acids. In other words, ATG will specify the amino acid methionine in almost every organism. Because of this, a gene that is transferred from one species to another will almost always code for a protein of the same amino acid sequence in the new species as in the original species from which the gene came. When we say that the genetic code is redundant, what we mean is that most amino acids are specified by more than one codon. For example, there are six different codons that all specify the amino acid leucine. When we say that the genetic code is non-ambiguous, what we mean is that each codon specifies only one amino acid. So, multiple codons can specify the same amino acid, but a given codon only specifies a single amino acid. When we say that the genetic code exhibits wobble, what we mean is that the anticodons of some tRNAs are able to base pair with multiple codons that have almost identical sequences. In cases where a tRNA can base pair with multiple codons, each of the codons the tRNA can base pair with will specify the same amino acid. So, wobble does NOT cause the genetic code to be ambiguous. A particular tRNA will always help insert the same amino acid into a polypeptide, its just that the tRNA may act at more than one codon that specifies that particular amino acid.

When trying to remember the difference between inducers and corepressors of repressor proteins, remember that these small molecules are NOT named because of how they affect the activity of the repressor protein.  Instead, they are named because of how they affect the rate of transcription of the genes in the operon(s) they help regulate.  So, inducers are called inducers because scientists used to think that they induce (i.e. turn on) expression of the genes in the operons they help regulate. We now know that what inducers actually do is to stabilize repressor proteins in their inactive form.  By stabilizing repressor proteins in their inactive form, inducers keep the repressor proteins from binding to the operator sequences and preventing transcription.  So, in a sense, inducers allow gene transcription to proceed by turning off a repressor.   Corepressors work in kind of the opposite way.  Corepressors stabilize the repressor proteins with which they interact in their active form.  So, when the correct corepressor is present in a cell, the repressor protein with which it interacts will bind to it and be stabilized in its active form.  The active repressor/corepressor complex will then bind the operator sequence, preventing transcription of the genes in the operon by RNA polymerase.  So, the name corepressor refers to the fact that corepressors help repress transcription of the genes in the operon.
